# WarmPaws - Pet Winter Care Platform 

## Overview
**WarmPaws** is a pet-focused website designed to provide winter care solutions for pets. The platform showcases a variety of pets and their photos, highlighting essential products and services to keep them healthy and comfortable during the cold season. WarmPaws helps pet owners take better care of their furry friends with personalized advice, clothing, accessories, and veterinary guidance.

---

## Features

### 1. User Authentication
- **Sign Up / Login** via email & password.
- **Google Login** for quick access.
- **Password Reset** for account recovery.
- Secure user sessions using Firebase Authentication.

### 2. Pet Profiles
- Browse various pets with photos and descriptions.
- Each pet has information about their winter care requirements.
- Users can view pet-specific recommendations for clothing, balm, beds, and more.

### 3. Winter Care Products
- **Clothing:** Coats, sweaters, and paw protectors for pets.  

- **Balm & Skincare:** Protect petsâ€™ skin and paws from dryness and cold.  

- **Beds & Blankets:** Ensure pets stay warm during harsh winters.  
- **Holiday Outings:** Tips and images on safe outdoor activities for pets in winter.

### 4. Vet Advice Section
- Detailed advice from professional veterinarians.
- Tips on feeding, grooming, and winter-specific care.
- Safety recommendations for outdoor activities.

### 5. User-Friendly Interface
- Responsive design for desktop and mobile.
- Intuitive navigation with clear sections for pets, products, and advice.
- Quick access to favorite pets and their care guides.

## Tech Stack
- **Frontend:** React, React Router, Tailwind CSS  
- **Backend / Authentication:** Firebase Authentication  
- **State Management:** React Context API  
- **Notifications:** react-hot-toast for success/error messages  
- **Icons & Graphics:** react-icons for UI elements
